# dret

Daily returns

### Usage

```python
dret [-p PERIOD]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| period | Period to select start end of the year returns | all | True | 3y, 5y, 10y, all |
